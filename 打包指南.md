# Android APK 打包指南

## 方案1：使用在线构建服务（最简单）

### GitHub + Actions 自动构建

1. 在 GitHub 创建新仓库
2. 上传整个 `TVPlayerAndroid` 文件夹
3. 在仓库创建 `.github/workflows/build.yml`:

```yaml
name: Build APK

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  build:
    runs-on: ubuntu-latest
    
    steps:
    - uses: actions/checkout@v3
    - name: Set up JDK
      uses: actions/setup-java@v3
      with:
        java-version: '17'
        distribution: 'temurin'
    - name: Grant execute permission for gradlew
      run: chmod +x gradlew
    - name: Build with Gradle
      run: ./gradlew assembleDebug
    - name: Upload APK
      uses: actions/upload-artifact@v3
      with:
        name: TVPlayer-debug.apk
        path: app/build/outputs/apk/debug/app-debug.apk
```

4. 推送代码，Actions 会自动构建
5. 在 Artifacts 下载 APK

## 方案2：使用 Appetize（在线运行APK）

1. 先通过方案1或其他方式获得 APK 文件
2. 访问 https://appetize.io
3. 上传 APK 文件
4. 获取在线运行链接

## 方案3：使用 WebView Gold（在线打包）

1. 访问 https://www.webviewgold.com/
2. 选择 Android
3. 输入相关信息
4. 上传 APK 或提供功能需求
5. 获取打包后的 APK

---

## 方案4：手动使用 Android Studio

### 安装 Android Studio

1. 下载: https://developer.android.com/studio
2. 安装过程会自动下载 SDK

### 导入项目

1. 打开 Android Studio
2. File → Open → 选择 `TVPlayerAndroid` 文件夹
3. 等待 Gradle 同步完成

### 构建 APK

1. Build → Build Bundle(s) / APK(s) → Build APK(s)
2. 等待构建完成
3. 点击 "locate" 查看 APK 位置

---

## 方案5：使用命令行（需要手动下载 Gradle）

1. 下载 Gradle: https://gradle.org/releases/
2. 解压到本地目录
3. 在项目目录执行:
   ```bash
   gradle wrapper
   gradle assembleDebug
   ```

---

## 注意事项

1. **Android 5.0+**: 本应用最低支持 Android 5.0 (API 21)
2. **网络权限**: 应用需要互联网权限访问直播源
3. **电视适配**: 已设置横屏模式，适合 TV 使用

---

## 获得帮助

如有问题，请检查:
1. Java 版本是否 >= 8
2. Android SDK 是否正确安装
3. 网络是否正常
